# Online-Ticket-Booking-System-Bus-train-Plane
## Project Description
This project is developed for the CSE327 Software Development course. 
The system allows users to book tickets for buses, trains, and airplanes through an online platform. 
Users can search routes, check schedules, select seats, and make bookings easily.

The system improves convenience by providing a centralized platform for transportation ticket booking.

## End Users
The end users of this system are:
- Passengers who want to book tickets
- Admin who manages routes and schedules
- Transport companies who provide transportation services

## Functional Requirements
- User registration and login
- Search bus/train/flight routes
- Ticket booking system
- Seat selection
- Payment processing
- Booking history

## Non-Functional Requirements
- User-friendly interface
- Fast response time
- Secure user authentication
- Reliable database storage
- System availability
